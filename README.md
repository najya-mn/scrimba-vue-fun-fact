# Vue 3 Fun Facts App

This is a simple fun fact app made with Vue 3.



## Technologies Used

* HTML
* CSS
* Vue.js 3

[Vue 3 Fun Facts App - Live Demo](https://najya-mn.github.io/scrimba-vue-fun-fact/)

![Fun Facts About Rin Okumura](/screenshots/fun-facts.png)



## Features

* Shows trivia about Rin Okumura using reusable Vue 3 components and v-for to loop over an array of facts. 
* Simple but clean layout with consistent spacing and typography that works well on both desktop and mobile devices. 
* New facts can be added by simply editing the array in the component, making the app maintainable and easy to extend.



## What I Learned

* Vue 3 fundamentals: Practiced using <script setup>, reactive refs, components, props, and template syntax (v-for, dynamic binding). 
* Component‑based thinking: Split the UI into small, reusable components (facts, header, footer) to keep the code organized and DRY. 
* Vite and builds: Learned how Vite builds the app (npm run build) and how to configure base in vite.config.js for sub‑folder hosting. 
* GitHub Pages deployment: Gained experience deploying a static Vue 3 + Vite app to GitHub Pages by moving dist/ into docs/ and setting the correct folder source. 
* CSS scoped styling: Used <style scoped> to keep styles local to each component, avoiding global CSS conflicts.
